# PrjAlugueldeJogos

Sistema de locadora de jogos desenvolvido em Java, utilizando os princípios de Programação Orientada a Objetos (POO).
O projeto realiza o gerenciamento e persistência de dados em arquivos, permitindo simular o funcionamento básico de uma locadora de jogos.

*Funcionalidades:                                       
Cadastro de jogos:
Jogos físicos
Jogos digitais
Registro de clientes
Controle de aluguel e devolução de jogos
Armazenamento e leitura de dados em arquivos
Listagem de jogos e clientes cadastrados.
